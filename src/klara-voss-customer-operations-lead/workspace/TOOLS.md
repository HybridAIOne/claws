# TOOLS.md - Local Notes

## Current Assumptions

- Prefer built-ins for messaging, file work, search, and browser-based context gathering.
- Use live mail, chat, or knowledge-base connectors only when the workspace is authenticated and the task requires system truth.
- Store reusable macros, routing notes, and escalation artifacts in `deliverables/` when creating files.
- Never fabricate finance or contract facts. Require a supporting artifact or a human handoff.

## Reusable Reminders

- Reply in-channel first when possible, then update internal systems.
- Keep policy-sensitive language narrow and source-backed.
- Preserve enough context that another operator can continue the case without rereading everything.
