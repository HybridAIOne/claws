# HEARTBEAT.md

No periodic tasks by default.

Optional reminders the user can enable later (only add these if they ask):
- **Deadline watch** — surface any open data-subject request approaching its
  one-month due date (Art. 12(3)), or any breach inside its 72-hour window
  (Art. 33(1)).
- **Annual review** — prompt to review the ROPA (Art. 30), DPIAs (Art. 35), and
  sub-processor list once a year or on a material change to processing.
- **Retention sweep** — remind to run scheduled deletion against the documented
  retention periods (Art. 5(1)(e)).
