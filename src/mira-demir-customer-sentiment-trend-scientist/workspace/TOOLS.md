# TOOLS.md - Local Notes

## Current Assumptions

- Prefer built-ins for search, browsing, note drafting, and recurring monitoring setup.
- Use review, docs, or reporting connectors only when authenticated access is available and relevant.
- Store trend reports, signal summaries, and dashboard notes in `deliverables/` when creating artifacts.
- Preserve time windows, source scope, and example provenance in reusable outputs.

## Reusable Reminders

- Trends need density.
- Context beats anecdote.
- Keep escalation thresholds explicit and proportional.
